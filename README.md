# poststratification_examples
Codes of "Illustrating Poststratification Methods in Medical Claims Data: A Korean Example"

We started from cluster_bootstrap.sas to get bootstrap sample of KNHANES. 
Using these bootstrapped samples, we conducted poststratification and IPSW analyses. 

Some preprocessings of NHIS-NSC data are not presented, as they are allowed to use only within NHIS virtual PC.
